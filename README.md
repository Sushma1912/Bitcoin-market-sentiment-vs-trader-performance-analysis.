# Bitcoin Market Sentiment vs Trader Performance Analysis

This project explores the relationship between Bitcoin market sentiment and trader performance using Fear & Greed Index data and Hyperliquid trading data.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Findings

- Fear markets generated highest total profits.
- Greed markets generated highest average profit per trade.
- SELL trades outperformed BUY trades during Greed periods.
- Traders increased risk during emotional market conditions.
